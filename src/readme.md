# W.O.P.R.

Es una aplicación escrita en Java que simula el acceso al W.O.P.R., un sistema militar automatizado diseñado para responder ante amenazas nucleares.
El usuario adopta el rol del único operador con acceso directo al sistema.
La historia arranca en una base militar de EE.UU., donde una IA clasificada pregunta: “¿Quieres jugar?”.
El programa combina narrativa interactiva y decisiones del jugador.
Su objetivo es recrear la tensión y el suspense de gestionar un sistema capaz de decidir el destino del mundo y hacer una reflexion sobre la relación actual que hay con la inteligencia artificial. 

## Tecnologias utilizadas

Java 21
IntelliJ IDEA

## Estructura del proyecto

`src/`

├──` Funciones/` Aqui se encuentran todas las funciones que desarrollan el juego

│   ├──` Main/` Aquí llamamos a las funciones para ejecutarlas y utilizamos un bucle while por si el usuario cuando pierde quiere volver a jugar sin necesidad de arrancar de nuevo el juego

### Cómo ejecutar el proyecto

1. Abrir el proyecto en IntelliJ IDEA.
2. Asegurarse de que está configurado el JDK correcto (21) 
3. Abrir la clase `Main.java`.
4. Clic derecho → Run 'Main'.
